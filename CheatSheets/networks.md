# Сети

## Модель OSI

 | Уровень OSI | Название | Среда (объекты) | Описание (для чего нужна) | Примеры протоколов с упором на безопасность |
 |-|-|-|-|-|
 | 7. Прикладной уровень | Application Layer | Приложения, данные | Обеспечение взаимодействия приложений | HTTPS, FTPS, S/MIME, DNSSEC |
 | 6. Уровень представления | Presentation Layer | Данные, форматы | Преобразование данных, шифрование | SSL/TLS, SSH, PGP |
 | 5. Сеансовый уровень | Session Layer | Сессии, соединения | Управление сессиями и соединениями | TLS, SSH, NetBIOS |
 | 4. Транспортный уровень | Transport Layer | Пакеты, порты | Обеспечение надежной передачи данных | TCP, UDP, SCTP, DTLS |
 | 3. Сетевой уровень | Network Layer | Пакеты, маршруты | Определение маршрутов и передача данных | IPsec, ICMP, IPv4, IPv6 |
 | 2. Канальный уровень | Data Link Layer | Кадры, MAC-адреса | Обеспечение надежной передачи данных по физическому каналу | Ethernet, PPP, MACsec |
 | 1. Физический уровень | Physical Layer | Биты, сигналы | Передача необработанных данных по физическому каналу | Ethernet, Wi-Fi, Bluetooth |
