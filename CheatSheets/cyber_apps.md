 | **Policy Management** | Управление политиками безопасности | Объекты: политики, правила <br> Технологии: централизованное управление, автоматическое обновление <br> Каналы утечек: все каналы, управляемые политиками |
 | **Incident Management** | Управление инцидентами безопасности | Объекты: инциденты, оповещения <br> Технологии: автоматическое реагирование, уведомления <br> Каналы утечек: все каналы, где произошли инциденты |
 | **Compliance Reporting** | Отчеты о соответствии требованиям безопасности | Объекты: отчеты, логи <br> Технологии: генерация отчетов, анализ данных <br> Каналы утечек: все каналы, управляемые политиками |
 | **User Activity Monitoring** | Мониторинг активности пользователей | Объекты: пользователи, события <br> Технологии: мониторинг активности, анализ поведения <br> Каналы утечек: все каналы, где происходит активность пользователей |
 | **Encryption** | Шифрование данных для защиты от утечек | Объекты: файлы, электронные письма, данные <br> Технологии: шифрование, управление ключами <br> Каналы утечек: электронная почта, облачные сервисы, USB-устройства |
 | **Watermarking** | Добавление водяных знаков для отслеживания данных | Объекты: документы, изображения <br> Технологии: добавление водяных знаков, отслеживание <br> Каналы утечек: электронная почта, облачные сервисы, печать |
 | **Behavioral Analysis** | Анализ поведения для выявления аномалий | Объекты: пользователи, события <br> Технологии: машинное обучение, анализ поведения <br> Каналы утечек: все каналы, где происходит активность пользователей |
 | **Data Masking** | Маскирование данных для защиты конфиденциальной информации | Объекты: данные, документы <br> Технологии: маскирование данных, анонимизация <br> Каналы утечек: базы данных, документы, электронная почта |
 | **Cloud DLP** | Защита облачных сервисов от утечек данных | Объекты: файлы, электронные письма, данные <br> Технологии: API-интеграция, мониторинг облачных сервисов <br> Каналы утечек: облачные хранилища, облачные приложения, веб-трафик |
